Можно подключить до 4 шаговых двигателей.

Основные параметры:

- диапазон напряжения работы 4.5-48 В, для разных каналов можно использовать разное напряжение;
- ток до 2,5 А;
- микрошаг до 1/256.

В качестве драйвера для двигателя используется чип **DRV8834S**.

## Схема внешних подключений

Для разных каналов можно использовать разное напряжение. Минусовой провод всех источников объединяется на модуле.

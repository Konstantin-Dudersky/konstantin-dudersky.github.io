Можно собирать как с моделью **Luckfox Lyra** [^1], так и с моделью **Luckfox Lyra Plus** [^2]. Посадочное место у контроллеров одинаковое, у модели **Plus** есть дополнительный интерфейс Ethernet.

Через интерфейс DSI можно подключать дисплеи с тач-скринами:
- Официальные дисплеи Raspberry Pi (https://www.raspberrypi.com/products/touch-display-2/)
- Дисплеи Luckfox (https://www.luckfox.com/index.php?route=product/category&path=228_229_237)
- Дисплеи Waveshare (https://www.waveshare.com/catalogsearch/result/?q=DSI)

Операционная система - Linux, собираемая с помощью Buildroot. Производитель Luckfox [предоставляет SDK](https://wiki.luckfox.com/Luckfox-Lyra/SDK-Image-Compilation) для сборки образа. При сборке также назначается фунциональность ножек контроллера (Rockchip Matrix).

При сборке модуля на контроллер загружается прошивка со всеми необходимыми настройками.

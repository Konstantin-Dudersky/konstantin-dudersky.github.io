Микроконтроллер подключается к чипам модуля через интерфейсы I²C и/или SPI. Для связи с контроллером используется шина CAN.

Два 12-пиновых FFC-коннектора используются для последовательного соединения соседних модулей в одну цепь.

Плата подключается к модулю через Pin Header с шагом 2 мм. Микроконтроллер подключается к чипам модуля через интерфейсы I²C и/или SPI. В разных модулях используется разная распиновка, для получения информации смотри схему каждого модуля.

На плате расположен CAN-трансивер **CA-IS3050** для подключения микроконтроллера к шине CAN. Используемые пины:

- CAN RX - GPIO0
- CAN TX - GPIO1

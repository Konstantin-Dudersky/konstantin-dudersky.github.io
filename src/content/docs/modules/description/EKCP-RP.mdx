На плате есть отверстия для закрепления как полноценной Raspberry Pi, так и укороченной Raspberry Pi Zero. Также можно использовать все возможные клоны, совместимые по 40-пиновому коннектору и креплению.

Ссылки на некоторые совместимые платы:

- полный форм-фактор:
    - Raspberry Pi 5 (https://www.raspberrypi.com/products/raspberry-pi-5/)
    - Luckfox Lyra Pi (https://www.luckfox.com/Luckfox-Lyra-Pi)
    - Luckfox Pico Pi (https://www.luckfox.com/luckfox-pico-pi)
- укороченный форм-фактор:
    - Raspberry Pi Zero W (https://www.raspberrypi.com/products/raspberry-pi-zero-2-w/)
    - Luckfox Lyra Zero W (https://www.luckfox.com/Luckfox-Lyra-Zero-W)
    - Luckfox Pico Zero (https://www.luckfox.com/Luckfox-Pico-Zero)

Необходимо правильно настроить функцию пинов на используемом одноплатнике.

Для связи между модулями используется шина CAN 2.0. В некоторых одноплатниках есть встроенный CAN-контроллер. Для моделей без встроенного CAN-контроллера (например, Raspberry Pi) на плате располагается внешний CAN-контроллер **MCP2515**, работающий через интерфейс SPI.

При установке одноплатника Raspberry Pi полного форм-фактора интерфейсы USB и Ethernet выходят за границу модуля. Также перекрывается одно отверстие M4 для крепления модуля.

При использовании форм-фактора Raspberry Pi Zero одноплатник не выходит за границы модуля.

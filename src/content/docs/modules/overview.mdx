Линейка модулей, по аналогии с модулями промышленных контроллеров. Протокол обмена данными между модулями - CAN2.0.

В чём отличие от китайской рассыпухи:

- Размеры всех плат одинаковы, можно аккуратно собрать на стойках M4 или в корпусе.

- Устройства подключаются через разъемные клеммники, а не под пайку - проще собирать и проверять схему.

- Понятная схемотехника, поставщики комплектующих - lcsc, chipdip.

- Спроектированы для ручного монтажа. Хорошая ремонтопригодность.

- Все модули имеют гальваническую изоляцию внутренней шины от периферийных устройств. Защита контроллера и других модулей.

В чём отличие от фирменных промышленных контроллеров:

- Стоимость - в разы меньше, чем фирменные контроллеры.

- Открытая схемотехника.

- Модули поставляются с написанной прошивкой, но можно написать и свою.

- Возможность встраивания в свой корпус.

Область применения:

- комплектущие блоков управления для серийных устройств

- прототипирование устройств

- домашняя автоматизация

## Внешние размеры и крепление

import Dimensions from "./dimensions.svg";

<Dimensions />

Размеры печатных плат модулей: 100 мм x 70 мм.

На всех печатных платах есть 4 отверстия для крепления стойками M4.

С одной стороны платы находится клеммник, который выходит за пределы печатной платы.

4 отверстия под стойки M2 нужны для крепления модулей светодиодной индикации (PMLD).

## Шина IBus

import IBus from "./ibus.svg";

<IBus />

Модули подключаются друг к другу через 20-пиновый FFC-кабель. В этом кабеле передаётся:

- Постоянное напряжение нескольких номиналов: 24В, 5В, 3.3В.
- Шина I²C - для связи контроллера (PMCP) с модулями ввода/вывода (PMIO).
- Шина CAN - для связи контроллера (PMCP) с модулями ввода/вывода (PMIO). Линии CAN гальванически изолированы от остальной схемы.
- Сигнал RST - выход сторожевого таймера (watchdog) [STWD100](https://www.st.com/en/reset-and-supervisor-ics/stwd100.html). Необходим для сброса выходов при остановке программы управления.

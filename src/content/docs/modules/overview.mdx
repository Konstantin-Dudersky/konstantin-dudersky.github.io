## Описание

Модули можно разделить на несколько групп:

| Группа | Описание |
|--------|----------|
| PMPS | блоки питания |
| PMCP | контроллеры |
| PMIO | дискретные и аналоговые входы/выходы |
| PMIF | интерфейсы |
| PMDR | драйверы двигателей |

## Шина IBus

Модули подключаются друг к другу через 20-пиновый FFC-кабель. В этом кабеле передаётся:

- Постоянное напряжение нескольких номиналов: 24В, 5В, 3.3В.
- Шина I²C - для связи контроллера (PMCP) с модулями ввода/вывода (PMIO).
- Шина CAN - для связи контроллера (PMCP) с модулями ввода/вывода (PMIO). Линии CAN гальванически изолированы от остальной схемы.
- Сигнал RST - выход сторожевого таймера (watchdog) [STWD100](https://www.st.com/en/reset-and-supervisor-ics/stwd100.html). Необходим для сброса выходов при остановке программы управления.

## Описание

<Schematic />

На плате установлен микроконтроллер **ESP32-C3** [^2], который опрашивает АЦП по интерфейсу SPI и отправляет данные во внутреннюю шину IBus через CAN-шину.

Перед опросом по шине SPI необходимо выбрать необходимый чип с помощью сигнала CS (Chip Select). Для этого микроконтроллер **ESP32-C3** по шине I²C управляет чипом **MCP23017** [^3].

Распиновка микроконтроллера **ESP32-C3**:

- SPI
  - MISO - GPIO 0
  - SCK - GPIO 1
- I²C:
  - SDA - GPIO 8
  - SCL - GPIO 2
- CAN:
  - RX - GPIO 10
  - TX - GPIO 7

**ESP32-C3** передаёт данные на CAN-шину через CAN-трансивер **CA-IS3050** [^4].

Для компенсации температуры холодного спая на плате установлен датчик температуры **TMP75**. Адрес датчика на шине I²C - 0x4C (A2=true, A1=false, A0=false).
